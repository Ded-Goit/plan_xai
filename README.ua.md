# 🛠️ Локальний запуск

### 1. Необхідні умови
Цей проект використовує **pnpm** як менеджер пакетів. Переконайтеся, що він у вас встановлений:
```bash
# Встановіть pnpm глобально, якщо у вас його немає
npm install -g pnpm
```

### 2. Встановлення
Встановіть залежності проекту та схваліть необхідні скрипти збірки (наприклад `msw`):
```bash
pnpm install
pnpm approve-builds
```

### 3. Запустіть сервер розробки
Запустіть локальний сервер у режимі розробки:
```bash
pnpm dev
```
Відкрийте [http://localhost:3000](http://localhost:3000) за допомогою браузера, щоб побачити результат.
